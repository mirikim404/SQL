# SQL 첫걸음 실습

> 책: SQL 첫걸음 | 20~30강 실습 파일

## 환경 설정

1. [MySQL 설치](https://dev.mysql.com/downloads/mysql/)
2. VSCode에서 확장프로그램 `MySQL (by cweijan)` 설치
3. VSCode 사이드바 DB 아이콘 → MySQL 연결
4. 각 `.sql` 파일 열고 실행

## 폴더 구조

```
sql-study/
├── 20강_COUNT/
├── 21강_집계함수/
├── 22강_GROUP_BY/
├── 23강_서브쿼리/
├── 24강_상관서브쿼리/
├── 25강_데이터베이스객체/
├── 26강_테이블DDL/
├── 27강_제약/
├── 28강_인덱스구조/
├── 29강_인덱스작성삭제/
└── 30강_뷰/
```

## 실습 순서

각 강의 폴더 안의 `practice.sql` 파일을 순서대로 실행하면 됨.
파일 상단에 테이블 생성 및 데이터 삽입 코드가 포함되어 있으므로
**위에서부터 순서대로 실행**할 것!

## 주의사항

- 각 파일 상단의 `DROP TABLE IF EXISTS` 로 기존 테이블을 초기화하므로 처음부터 실행 가능
- 에러 발생 의도 코드는 `--` 주석 처리되어 있음. 직접 주석 해제해서 에러 확인 가능
- MySQL 기준으로 작성됨 (Oracle은 일부 문법 다를 수 있음)
